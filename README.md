НФТ ДЛЯ СИСТЕМЫ ЗАПИСИ К ВРАЧУ
1. Доступность
Tag: Непрерывность работы системы записи при пиковых нагрузках и сбоях инфраструктуры.
Ambition: Сервис записи к врачу должен быть доступен для пациентов круглосуточно, минимизируя время простоя при аппаратных сбоях. 
Scale: Процент времени в течение календарного месяца, когда пациент может успешно загрузить главный экран и отправить запрос на запись. 
Meter: Автоматизированный мониторинг (например, Pingdom/Uptime Robot), отправляющий GET-запросы к API записи каждые 10 секунд. Расчет по формуле: (время ответов с кодом 200 / общее время в месяце) × 100. 
Goal: Не менее 99.5% времени в месяц (около 3.6 часов простоя в месяц).
Stretch: Не менее 99.9% времени в месяц (около 43 минут простоя в месяц).
